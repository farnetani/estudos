# Header e Footer Fixed

Fonte: https://cssreset.com/creating-fixed-headers-with-css/

Crie um arquivo chamado: menu.html com o código abaixo e teste rolar a página.

.footer {
    height:50px;
    background:#2a2f50;
    border:1px solid #CCC;
    width:960px;
    margin:0px auto;
    color: #fff;
}


</style>
</head>
